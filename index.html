Инструкция по установке

1. Настройка сервера

  1. Скачайте и установите ngrok
  2. Запустите сервер:
    python server.py
  3. Откройте ngrok и введите(например):
    ngrok tcp 5555
  4. Скопируйте полученный адрес и порт из строки(возможный пример):
    tcp://8.tcp.ngrok.io:12345

2. Настройка клиента

  • При запуске клиента вам потребуется ввести:
    • Сначала адрес сервера (например: 8.tcp.ngrok.io)
    • Затем порт (например: 12345)

    Важно

    • Запускайте сначала сервер, затем клиент
    • Адрес меняется при каждом запуске ngrok
    • Можно подключить несколько клиентов
